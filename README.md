# MineSweeper
A playground for the Minesweeper game written in C++. The rules are here https://en.wikipedia.org/wiki/Minesweeper_(video_game)

## Running the game

The game can be run via the g++ compiler on Linux:

```
g++ game.cpp
./a.out
```
